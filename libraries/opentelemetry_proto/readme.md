These are the opentelemetry protobuf compiled via [nanopb](https://github.com/nanopb/nanopb) files from;

    https://github.com/open-telemetry/opentelemetry-proto

It is currently based on v1.9.0 of opentelemetry-proto.

The compiled nanopb files can be used to generate OTLP protobufs messages on microprocessors.
